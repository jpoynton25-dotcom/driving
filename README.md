# Driving Website Starter

Lean static HTML starter optimized for speed and SEO.

## Included
- Semantic, accessible premium `index.html`
- Multi-page conversion site:
  - `theory-test.html`
  - `hazard-perception.html`
  - `practical-test.html`
  - `free-resources.html`
  - `pass-pack.html`
  - `driving-lessons.html`
  - `intensive-courses.html`
- Lightweight CSS and JS (`styles.css`, `main.js`)
- Technical SEO files: `robots.txt`, `sitemap.xml`, `llms.txt`
- JSON-LD (`DrivingSchool` schema)

## Next actions
1. Replace `example.com` values with your real domain.
2. Replace brand/content details with your real business information.
3. Add optimized images in `assets/img/` (WebP/AVIF preferred).
4. Deploy on a static host (Cloudflare Pages, Netlify, Vercel, GitHub Pages).
